files for the SP26 PBHLTH142 Data Project

dataset adapted from NHANES 2017-2018, filtering obtained via Stat C131A

to download the dataset via datahub, run the following in the **terminal**:

```cd; wget https://raw.githubusercontent.com/kittyjosh111/pbhlth142-dataproject/refs/heads/main/dataset/nhanes_obesity.csv```

to download this entire repository via datahub, run this in the **terminal**:

```cd; git clone https://github.com/kittyjosh111/pbhlth142-dataproject.git```
